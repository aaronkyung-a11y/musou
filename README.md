# 無雙記 · 무쌍기 (Musou)

삼국지 무장으로 즐기는 **뱀파이어 서바이벌류 액션 로그라이크**. 단일 HTML 파일, 오프라인 동작, 설치형 PWA.

## 🎮 플레이

**Play**: https://aaronkyung-a11y.github.io/musou/

가로 모드 전용. 모바일에서 홈 화면에 추가하면 전체화면 앱처럼 실행됩니다.

## 조작

- **이동**: 화면 왼쪽을 드래그 → 가상 조이스틱
- **공격**: 무기 자동 발동 (무장마다 고유)
- **레벨업**: 적 처치 → 경험치 구슬 수집 → 강화 3택
- **필살기**: 우하단 게이지가 차면 발동, 화면의 적을 일소
- **목표**: 300초 생존

## 무장

| 무장 | 무기 | 필살기 |
|------|------|--------|
| 관우 | 청룡언월도 (회전 참격) | 청룡참 |
| 조운 | 용담창 (관통) | 칠진돌파 |
| 장비 | 장팔사모 (부채꼴) | 당양대갈 |
| 여포 | 방천화극 (고속 회전) | 무쌍난무 |
| 황충 | 철태궁 (원거리 화살) | 연노난사 |
| 태사자 | 쌍철극 (전방 연격) | 쌍극선풍 |

## 기술

- 순수 HTML5 Canvas 2D, 외부 라이브러리 없음
- Service Worker 기반 오프라인 캐시
- 세로 감지 시 가로 회전 안내
