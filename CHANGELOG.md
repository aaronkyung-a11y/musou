# 일기토 (一騎討) — CHANGELOG

기존 삼국지 전략 프로젝트(sam-v1.0)의 자산 — 무장 스탯, SVG 초상화, 일기토 이펙트 레이어(슬래시·임팩트 플래시·집중선·파티클·shake) — 를 재사용해 **리듬 연타 액션 일기토** 독립 게임으로 재구성.

## v0.1 (2026-07-12)
- **리듬 연타 코어**: 노트가 판정선(금색 원)으로 흘러오면 하단 입력 영역을 조작. 탭/스와이프/홀드 3종 노트.
- **판정 시스템**: PERFECT / GOOD / MISS. 거리 기반 판정창 + 노트 타입 일치 여부.
- **스탯 반영 (기존 77명 중 50명 이식)**:
  - 무력(武) → 공격 데미지 배율
  - 통솔(統) → 최대 체력 및 판정 관용창 폭
  - 적 무력이 높을수록 노트 속도·빈도 증가 (난이도)
- **콤보 시스템**: 연속 히트 시 데미지 증가(합당 +5%, 최대 20합). 5콤보 이상에서 집중선 발동.
- **이펙트 재사용**: PERFECT→참격 슬래시+임팩트 플래시, 피격→shake+파티클, 콤보→집중선.
- **SVG 초상화**: 진영색 + 이름 시드 기반 얼굴/투구/수염 절차 생성.
- 단일 파일 HTML, 오프라인, 모바일 세로 최적화.

### 알려진 한계 / 다음 후보

## v0.2 (2026-07-12) — 로그라이크 전환
리듬 연타 전투는 유지하되, Jurassic Survival 로그라이크의 검증된 구조(유물·웨이브·엘리트·강화선택)를 얹어 **일기토 로그라이크**로 재구성.

- **런(RUN) 구조**: 무장 1명으로 연속 일기토. 승리마다 강화 1택, 패배 시 런 종료. "얼마나 멀리 가는가"가 목표.
- **유물(RELIC) 시스템 13종**: 4등급(커먼/레어/에픽/레전더리). 공격·체력·판정·콤보·흡정·크리·선제·회복 등. 승리 후 3장 중 1택(a/b/c), 효과는 런에 누적 스택.
  - 적토마(레전더리): 전 데미지+25%, 노트속도-10%
  - 흡정의 검(에픽): PERFECT마다 피해의 15% 회복
  - 무신의 감각(레어): PERFECT 데미지+30% 등
- **적 스케일링**: 스테이지당 체력·공격 +14%. 3전부터 엘리트(변이) 20% 등장, 5의 배수는 보스(대장).
- **엘리트/보스 보상**: 처치 시 EPIC 이상 유물 확정.
- **체력 유지**: 전투 간 체력 비율 유지 → 자원 관리 요소. 일부 유물이 승리 시 회복.
- **HUD**: 상단에 현재 스테이지 + 보유 유물 아이콘 바.
- **선제·크리 연출**: 토스트 알림 추가.

## v0.3 (2026-07-12) — 뱀서류 전면 재설계 ⚔️ (musou-v0.3.html)
리듬 방식 완전 폐기. Vampire Survivors식 **가로 모드 액션 로그라이크**로 재구축. 제목 「無雙記(무쌍기)」.

- **가로 모드 전용**: Canvas 2D 실시간 렌더. 세로 시 회전 안내 표시.
- **조작**: 화면 왼쪽 드래그 = 가상 조이스틱 이동. 무기는 전부 자동 발동.
- **무장 6명 · 고유 무기**:
  - 관우/여포 = 회전 참격(spin), 장비/태사자 = 부채꼴 강타(cone),
  - 조운 = 관통 창(pierce), 황충 = 원거리 화살(shoot)
  - 무력→대미지, 통솔→체력에 반영
- **필살기**: 처치로 게이지 충전 → 우하단 버튼. 발동 시 풀스크린 플래시 + 기술명 연출 + 화면 전체 적 일소(수백 마리 처치 가능). 무장별 고유 필살(청룡참/무쌍난무/연노난사 등).
- **웨이브 스케일링**: 시간이 지날수록 스폰 가속·다중 스폰·적 강화. 정예(엘리트) 확률 등장(체력바 표시).
- **레벨업 로그라이크**: 처치→경험치 구슬 드랍→수집→레벨업 시 강화 3택(a/b/c). 무력/속공/범위/이속/체력/재생/관통/흡혈/급소 등 11종, 등급별 가중치.
- **승리 조건**: 300초 생존.
- **연출**: 피격 플래시, 처치 파티클, 데미지 텍스트(크리 강조), 필살기 대량 파티클.
- **밸런싱**: 초반 XP 곡선 완만화 + 획득 범위 120 → 약 10초 첫 레벨업, 이후 꾸준한 성장(headless 60초 시뮬 기준 Lv5 도달).

## v0.4 (2026-07-12) — 스킬·기마·병종·적장·박진감 (musou-v0.4.html)
대규모 액션성 강화 업데이트.

- **스킬 버튼 3종**: 기본공격 자동 발동 유지 + 우하단에 기술1(파랑)·기술2(빨강)·필살기(금색) 버튼. 무장별 고유 효과와 쿨다운.
  - 관우: 참마도(광역회전)/의기충천(3초 무적+공격↑), 조운: 난격(다연장)/질풍(순간이동돌진),
    장비: 단월(넉백강타)/호통(기절), 여포: 극무(고속연타)/패기(이속·공속↑),
    황충: 관통사/난사, 태사자: 연격/맹진(관통돌진)
- **주인공 기마화**: 적토마 탄 무장으로 그림. 다리 애니메이션, 무적 버프 시 청색 오라, 질풍 버프 시 말 붉어짐.
- **적 병종 구분**: 🐎기병(빠른 돌격) · 🏹궁병(거리 유지하며 원거리 사격, 적 투사체) · 보병(창). 각기 다른 외형·행동.
- **적 대량 증가**: 스폰 간격 단축(0.55→0.06s), 시간당 배치 증가(2+time/22마리씩). 화면을 가득 메움.
- **적장 등장 연출**: 15초 간격으로 적 장수(하후돈·장료·허저·안량 등 10명) 등장. 화면 밴드 슬라이드 + 이름 + 대사("눈 하나 잃었어도 너 따위는 벤다!") + 2.2초 일시정지. 큰 기마 + 깃발 + 이름표 + 체력바.
- **박진감 이펙트**:
  - 크리티컬 히트 → 순간 슬로우모션(0.35x) + 줌인(1.35x) + 붉은 비네트 + 진동
  - 필살기 → 슬로우(0.3x) + 줌(1.4x) + 화면 플래시 + 기술명
  - 적장 처치 → 강한 슬로우(0.25x) + 줌(1.5x) + "○○ 격파!" + 화면 흔들림
  - 카메라 시스템: 플레이어 중심 줌 + 흔들림(camShake)
- **캐릭터 선택 글자 깨짐 수정**: vh 폰트→px 고정, 3열 그리드, text-overflow:ellipsis로 잘림 방지.
